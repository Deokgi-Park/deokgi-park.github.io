# Paperclip 테스트

> **Paperclip**
>
> AI Agent를 고용한다는 가정 하에 오케스트레이션 하여 회사와 같이 운영가능한 오픈소스 플랫폼

해당 오픈소스를 활용해서 작은 게임 회사를 만들어 보고 싶었다.

다음 명령어를 터미널에 입력하는 것으로 퀵스타트가 가능하다

```bash
npx paperclipai onboard --yes
```

하필 4월 6일인 오늘 버그가 있어서
소스의 일부를 임시로 수정할 필요가 있었다.

```text
Paperclip server failed to start.
require() cannot be used on an ESM graph with top-level await.
Use import() instead. To see where the top-level await comes from,
use --experimental-print-required-tla.
```

일부 라이브러리에 종속성 문제가 있어서 발생한 문제로 보인다.

추후 패치된 버전을 따로 설치하기로 하고 해당 `parsers.js` 파일에 import 된 부분을 주석 처리 후 더미 변수를 만들어서 동작은 되게 수정했다.

```javascript
//parsers.js

//기존 변수 주석
// const {
//   resolve: resolveColor,
//   utils: { cssCalc, resolveGradient, splitValue }
// } = require("@asamuzakjp/css-color");

//변수들을 더미로 생성...
const resolveColor = () => null;
const cssCalc = () => null;
const resolveGradient = () => null;
const splitValue = () => [];
const utils = { cssCalc, resolveGradient, splitValue };
const resolve = resolveColor;
```

위와 같은 작업 후 회사명과 사용할 AI(claude, codex) 등을 선택하면 나만의 작은 회사를 만들 수 있었다.

# 후기

> 쓸만한 게 맞을까...?

처음 조직도가 구성되고 이것 저것 시켜보는 재미는 있었다. 아직 AI 스킬이 미숙해서인지 토큰만 왕창 먹고 결과물이 미흡한 괴물이 탄생하게 되었다... ㅜㅜ

![AI가 생성한 미흡한 결과물](/assets/images/posts/2026-04-04-260406_일기/image-1.png)

인코딩이 깨지는 증상도 있었고
(터미널 캐릭터 셋이나 해당 툴이 사용 하는 PostgreSQL의 캐릭터 셋 등의 문제로 보인다.)
이를 해결하기 위해 다시 질문하는 것으로 토큰 소비를 상당히 하게 되는 모습을 보았다.

간단한 문제는 토큰을 아껴야될 필요성이 보이고 AI 스킬, 훅 같은 것을 잘 설정해야될 것 같다. 잘모르는 분야니 이 또한 공부 해야될 것 같다.

게임에 필요한 캐릭터를 만들고 싶어서 부탁해보기도 했는데 클로드가 기본인 설정이라 이미지 생성 시 파이썬을 활용해서 만드는 시도를 보았다.

![파이썬으로 생성한 게임 캐릭터](/assets/images/posts/2026-04-04-260406_일기/image-2.png){: style="height: 300px;" .align-center}

<p style="font-size:0.5em; text-align:center;">너무 처참한 퀄리티......</p>

퀄리티가 너무 처참했다. 파이썬 라이브러리로 뭔가 막 그리던 거 같은데 위와 같은 퀄리티가 나왔다. 퀄리티를 해결해달라하니 별도의 일러스트 인원을 스스로 고용하는 모습을 볼 수 있었다. NOVELAI라는 별도의 AI를 활용하게끔 되있는 거 같은데 API 키 설정을 해달라고해서 일딴 패스했다.

![Paperclip으로 구성된 회사 조직도](/assets/images/posts/2026-04-04-260406_일기/image-3.png)

이런 식으로 회사가 만들어졌으나... 실제로는 AI 마다 별도의 스킬 교육이 필요할 것 같다. '어떤 라이브러리를 활용해라!', '너는 이 스킬을 써라!'

아직은 잘모르는 분야이고 토큰이 마구마구 빠지는 점이나 사실 혼자 사용하는데 이런 식으로 오케스트레이션 해도 결국 전부 관리포인트가 되는 점이 조금 애매하게 느껴지긴 했다.

AI관련 스킬이 늘면 더 좋은 설정으로 잘 쓸 수 있을까?
추후에 다시 도전해봐야될 것 같다.
